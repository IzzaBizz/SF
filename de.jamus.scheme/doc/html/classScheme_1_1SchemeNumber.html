SchemeBoolean Scheme.SchemeNumber.numberp ( )

Checks if this scheme object is a number object and returns a SchemeBoolean. Only the SchemeNumber class should return true, all other classes should return false!

Returns:
a SchemeBoolean with value 'true', if this scheme object is from type SchemeNumber, else 'false'.
