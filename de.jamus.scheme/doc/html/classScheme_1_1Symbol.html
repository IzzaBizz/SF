SchemeBoolean Scheme.Symbol.symbolp ( )

Checks if this scheme object is a Symbol object and returns a SchemeBoolean. Only the Symbol class should return true, all other classes should return false!

Returns:
a SchemeBoolean with value 'true', if this scheme object is from type Symbol, else 'false'.
