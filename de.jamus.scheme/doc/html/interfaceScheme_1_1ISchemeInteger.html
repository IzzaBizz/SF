SchemeBoolean Scheme.ISchemeInteger.integerp ( )

Checks if this scheme object is an integer object (a subtype of number object) and returns a SchemeBoolean. Only the SchemeInteger class should return true, all other classes should return false!

Returns:
a SchemeBoolean with value 'true', if this scheme object is from type SchemeInteger, else 'false'.
See also:
class SchemeInteger and class SchemeNumber.
